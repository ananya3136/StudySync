# 🚀 StudySync

A full-stack real-time study collaboration platform where students can chat, share files, and collaborate in groups.

---

## ✨ Features

* User Authentication (JWT)
* Study Groups
* Real-time Chat (Socket.io)
* Chat History (MySQL)
* File Upload (Multer)
* Email Notifications (Nodemailer)

---

## 🛠 Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MySQL (Sequelize)
* **Real-time:** Socket.io

---

## 🚀 Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```

---

## 📌 Overview

StudySync enables students to collaborate through real-time messaging, group discussions, and resource sharing in a structured environment.
